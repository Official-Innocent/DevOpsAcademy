# CamSync GMT

A real-time GMT clock and timezone reference tool built
for photographers who need to sync cameras on set.

## Features
- Live GMT clock with millisecond precision
- 20+ timezone options with city names
- No refresh needed — updates every 50ms
- Clean dark UI built for on-set use
- Fully containerised with Docker

## Run it instantly

You need Docker installed. Then:

    git clone https://github.com/Official-Innocent/DevOpsAcademy.git
    cd DevOpsAcademy/docker-lab
    docker compose up -d

Open your browser at http://localhost:5000

## Stop it

    docker compose down

## Tech stack
- Python 3.11 (stdlib only — no frameworks)
- Docker + Docker Compose
- GitHub Actions CI/CD (coming soon)
